Create target data directory
--------------------
make mktargetdir

Download data
--------------------
make download

Use Sampled data
--------------------
make sample

Use Original data
--------------------
make origin

Recommend Command Line
--------------------
If it is first time to setup data, I recommand follow instruction:
    1. make mktargetdir
    2. make download
    3. make origin
